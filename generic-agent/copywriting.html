## GenericAgent — 让 AI 真正操控你的电脑

我们每天都在折腾 AI Agent，但大多数框架装完就跑不起来——依赖一堆、配置复杂、最可怕的是：它根本记不住你是谁。

GenericAgent 改变这一切：3K 行代码，每次任务自动沉淀 Skill，越用越强，让 AI 真正成为你的专属助手。

**🔄 自我进化的 Agent，听起来像吹牛？**

真不是。每次你让它完成一个任务，它都会把执行路径自动固化成 Skill，写进记忆层（L3）。下一次再说同样的需求，它直接调用，不再重复摸索。

"监控股票并提醒我"——第一次：安装 mootdx、写脚本、配定时任务、调试。第二次开始：一句话启动。

用几周后，你的 Agent 拥有一套完全属于你自己的技能树，没有任何两个人的技能树是一样的。

**📦 3K 行代码，vs OpenClaw 53 万行**

你没看错，GenericAgent 核心只有约 3000 行代码，Agent Loop 只有约 100 行。OpenClaw 要 53 万行，Claude Code 同样体量庞大。

结果是什么？pip install 三个包，填个 API Key，python launch.pyw，直接跑起来。没有 Docker，没有多服务编排，没有复杂的配置文件。

**💪 真实浏览器，不靠无头模式**

大多数 Agent 框架用无头浏览器或者沙箱，登录状态丢失，每次都要重新登录。

GenericAgent 注入真实 Chrome，Cookie 和登录态完整保留。你平时怎么用浏览器，它就怎么用浏览器——而且还能操控你的鼠标键盘。

**🔗 全模型支持：Claude / Gemini / Kimi / MiniMax**

不支持你用的模型，再强也没用。GenericAgent 全都支持，一个框架换模型不改代码。

**⚡ 极致省 Token：<30K vs 200K~1M**

上下文窗口不到 30K，是其他 Agent 的零头。分层记忆（L0~L4）让关键信息始终在场，噪声少、幻觉低、成本低一个数量级。
